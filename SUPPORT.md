# Support

## Expectations

Import To Planner is a **solo spare-time project**. There are no guaranteed response times or SLAs. Support is provided on a best-efforts basis when time allows.

---

## Reporting a Bug

If you have found a bug, please [open a GitHub Issue](https://github.com/markheydon/import-to-planner/issues/new/choose) using the **Bug Report** template. Include as much detail as possible:

- Steps to reproduce the problem.
- Expected vs actual behaviour.
- .NET SDK version and operating system.
- A sample CSV file that demonstrates the issue (redact any sensitive data).

---

## Asking a Question or How-To

For general questions, usage questions, or how-to requests:

- [Open a GitHub Issue](https://github.com/markheydon/import-to-planner/issues/new/choose) with as much context as you can provide.
- If GitHub Discussions are enabled on this repository in the future, that will be the preferred place for open-ended questions.

---

## Reporting a Security Vulnerability

**Do not open a public issue for security vulnerabilities.**

Please read [`SECURITY.md`](.github/SECURITY.md) and use [GitHub's private vulnerability reporting](https://github.com/markheydon/import-to-planner/security/advisories/new) to disclose the issue privately.

---

## Microsoft Graph, Planner, and Entra ID Issues

This project calls the Microsoft Graph API to interact with Microsoft Planner. If you are experiencing problems that appear to be platform-level (Graph API errors, Planner service issues, Entra ID authentication failures), please consult:

- [Microsoft Graph documentation](https://learn.microsoft.com/graph/overview)
- [Microsoft Planner developer documentation](https://learn.microsoft.com/graph/planner-concept-overview)
- [Microsoft Entra ID documentation](https://learn.microsoft.com/entra/identity-platform/)
- [Microsoft support](https://support.microsoft.com)

Issues that are caused by the Microsoft platform rather than this application are outside the scope of this project's support.

---

## No Commercial Support

This is not a commercial product. No paid support, enterprise SLAs, or professional services are available.
