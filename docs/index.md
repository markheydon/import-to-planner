Import To Planner helps you move tasks from a CSV file into Microsoft Planner with a guided, step-by-step flow.

## Who this is for

This guide is for people who use Microsoft 365 at work or school and need a quick way to import task lists into Planner.

If you are setting up and running your own instance of the app, use the separate self-hosted guide.

## Start here

- Getting started: prerequisites, access, and sign-in expectations.
- CSV format: required headings, accepted values, and examples.
- Import workflow: what each step does and what the final report means.
- Troubleshooting: fixes for common sign-in, validation, and import issues.
- FAQ: short answers to common questions.
- Privacy and security: what data is used and how permissions work.
